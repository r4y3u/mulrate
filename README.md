# MulRate v2.0.0 alpha.3

乗法に特化したレート制トレーニングアプリです。v1.0.0 beta.19を基礎に、初回固定プロフィール、記録・ランキング、手書きパッド上の正誤表示、Supabase接続基盤を段階的に追加しています。

## alpha.3の主な更新

### 画面導線

- 初期画面から「プロフィール」「設定」の大きなボタンを削除。
- プロフィール画面は、初回起動時と全データリセット直後だけ表示。
- プロフィール決定後は、全データをリセットしない限りプロフィール画面へ戻らない。
- 上部右側は状況に応じて自動切替。
  - 回答中: `Ⅱ` ポーズ。
  - ポーズが不要な画面: 控えめな `⚙` 設定。
  - 初回プロフィール設定中: どちらも非表示。
- 設定画面を独立画面からオーバーレイへ変更。
- 設定オーバーレイは `× 閉じる` またはEscapeキーで閉じ、開く直前の画面をそのまま復元。

### プロフィール

- ニックネームは1〜12文字。保存後は全データリセットまで変更不可。
- Unicode正規化、ひらがな・カタカナの同一視、区切り記号を使った回避、代表的な英字置換を考慮して禁止語を判定。
- 短い英単語は完全一致に限定し、無関係な名前への過剰反応を抑制。
- 運営・管理者を装う表示名も拒否。
- ランキング公開への同意は任意。初回に同意しなかった場合のみ、設定オーバーレイから後で同意可能。
- 一度同意すると、全データリセットまで解除不可。

### 記録・ランキング

- 接続後は現在順位を `○位／○○位中` 形式で表示。
- ランキング枠をクリックすると1〜200位をオーバーレイ表示。
- 通信失敗時の送信データを端末内キューへ最大20件保持し、次回起動・オンライン復帰・ランキング表示時に再送。
- 同意前の過去記録は自動で遡って送信せず、同意後に完了したセットから送信。

### オンライン保護

- 端末ごとに公開しない秘密トークンを生成。
- サーバーにはSHA-256ハッシュだけを保存し、同じ端末IDの記録更新と削除に照合。
- 端末IDだけを知る第三者による上書き・削除を防止。
- プレイヤー単位で1分30件の基本送信制限を追加。
- Edge Functionの許可Originを環境変数で限定可能。
- セッション重複、問題・回答・正誤・時間・ダイジェスト・レート前後関係の基本検証を継続。

### 手書きパッド

- パッドを開いたまま回答した場合、パッド上へ大きな `○` または `×` を表示。
- ナビウィンドウ上配置とキーパッド上配置の両方に対応。

## 現在のオンライン動作

`online-config.js` は空欄です。配布状態では外部通信は発生しません。接続方法は `SUPABASE_SETUP.md` を参照してください。

サーバーは問題記録の整合性と端末認証を検証しますが、レート式全体のサーバー再計算は未完です。本番公開ランキングでは、送信されたレート値を信用せずサーバー計算値だけを採用する工程が必要です。

## 実行方法

`index.html` をブラウザで開きます。通常の学習機能は外部ライブラリを必要としません。

GitHub Pagesへ配置する場合は、次のファイルを同じ階層へ置きます。

- `index.html`
- `style.css`
- `app.js`
- `name-filter.js`
- `online-config.js`
- `online-adapter.js`

## データ移行

次の保存キーを検出すると、alpha.3形式へ自動移行します。

- `mulrate_v2_0_0_alpha2_state`
- `mulrate_v2_0_0_alpha1_state`
- `mulrate_v1_0_0_beta18_state`

移行時に端末秘密トークンを新規生成します。alpha.2以前のオンライン試験データが残っている場合、`002_player_auth_and_rate_limit.sql` 適用後の最初の正常送信で端末トークンが登録されます。
