# MulRate v1.0.0 beta.15

100段階カリキュラム版。beta.15では、熟達者領域のレート上昇を再調整し、F「得意・高速安定型」が100時間で最大レート `99999999` と学習済み100段階コンプリートに到達するようにした。

## beta.15 の主な変更

- 高難度・熟達者領域のレート倍率を強化
- 高レート帯でも伸びが止まりすぎないよう抑制式を調整
- 最終段階で高速・高正答を維持した場合、最大レートへ収束しやすくした
- 高難度段階の学習済み解放条件を微調整
- 通過済み段階の学習済み復元をセッション中にも反映
