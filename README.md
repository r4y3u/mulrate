# MulRate v1.0.0 beta.16

かけ算の計算力を鍛えるレート制トレーニングアプリです。静的HTML/CSS/JavaScriptのみで動作します。

## beta.16 changes

- スマホの問題画面で数式・解答欄が見えやすくなるよう、問題画面時の上下配分とゲームカード内の余白を調整。
- リザルト画面の「もう一度」を「スタート」に変更。
- 手書きパッド表示中は上部の「メモを開く／閉じる」ボタンを非表示化し、パッド領域を拡大。
- 手書きパッドのオーバーレイ時は「操作盤の配置」をグレーアウト。
- テンキー配置説明を1行に収まりやすい短文へ変更。
- 初期画面の最高レートカウントアップをタイマー基準に変更し、環境設定に左右されにくく調整。
- 正解演出のチェックマークを削除。誤答時は数式にも控えめな演出を追加。
- 学習済み画面に `n / 100` の解放度を控えめに表示。各類型に番号を表示。
- 手書きパッドに筆算用の被乗数・乗数を自動表示。消去時も元の式は残る。
- 筆算が必要な高負荷問題では、進行・レート判定用の時間基準をやや緩和。
