# MulRate v1.0.0 beta.18

乗法に特化したレート制トレーニングアプリです。

## beta.18の主な更新

- 問題回答後も手書きパッドを自動で閉じないようにした。
- 手書きパッド内の筆算テンプレートは問題ごとに更新し、ユーザーの筆記だけ消せる状態を維持した。
- 「消す」「閉じる」または「メモを開く」の近くに、入力済みの数字を控えめに表示するようにした。
- 左手用では、手書きパッド名を右側、操作ボタンを左側に配置した。
- UIコンセプト「思考を邪魔しない」「視覚負荷を抑える」「感覚をサポート」に合わせ、表示追加は小さく低刺激にした。

## 実行方法

`index.html` をブラウザで開くだけで動作します。外部ライブラリは不要です。
