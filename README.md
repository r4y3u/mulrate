# MulRate v1.0.0 beta.10

かけ算の技能を高め、知識の構造化を目的にした、ブラウザ単体動作のレート制トレーニングアプリです。

## beta.10 の主な変更

- カリキュラムを「乗法」中心に再編成しました。
- わり算・分数以降の内容は MulRate 本体から外しました。
- 将来のわり算領域は `DivRate` として別アプリ化する前提にしました。
- `KUKU_MIX_11_20` と `KUKU_MIX_0_20` を `M2D1_NC` 直前に移動しました。
- `M2D1_NC` の前に `何十×1けた` を追加しました。
- `M3D1_NC` の前に `何百×1けた` を追加しました。
- 2けた・3けたに10や何十をかける導入を整理しました。
- 3けた×2けたまでの整数乗法を追加しました。
- 小数は乗法のみ扱う方針にし、`0.?×1けた` から丁寧に導入する段階を追加しました。
- 小数の答えは、小数点を答え欄にあらかじめ表示し、学習者は各位の数字だけを入力する方式に変更しました。
- テンキー・キーボード入力から小数点入力を削除しました。

## 構成

- `index.html`：画面構造
- `style.css`：UI・配色・レスポンシブ
- `app.js`：出題、進行、レート、保存、入力処理
- `CURRICULUM.md`：実装済み学習段階一覧
- `DESIGN.md`：設計メモ
- `BALANCE.md`：バランス調整メモ
- `dev/simulate-balance.js`：モデルシミュレーション

## 実行方法

`index.html` をブラウザで開いてください。外部ライブラリ、サーバー、ビルドは不要です。

## 注意

beta.9以降ではカリキュラム順序を大きく変更しています。beta.8以前の保存データを使うと、進行位置が想定とずれる可能性があります。開発検証では、必要に応じて「学習データをリセット」を使ってください。
