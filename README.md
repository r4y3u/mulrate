# MulRate v1.0.0 beta.13

乗法技能を100段階で育てるブラウザ学習アプリです。

## beta.13 の主な変更

- 2けた×1けた、3けた×1けた、2けた×2けた、3けた×2けたを「繰り上がり」と「積の桁数」の両面で細分化
- 11〜20の段（1〜10）は3けた×1けた混合の後へ移動
- 乗数11〜20の拡張九九は3けた×2けた混合の後へ移動
- 89〜100に熟達者向け段階を追加
- 全体を100段階に固定
- 小数の答えは小数点を先に表示し、数字だけを入力

## ファイル

- `index.html`：起動ファイル
- `style.css`：低刺激の緑地・白文字UI
- `app.js`：出題、進行、保存、レート計算
- `CURRICULUM.md`：100段階一覧
- `dev/simulate-balance.js`：モデルシミュレーション

beta.11以前とは学習順序が異なるため、保存キーを更新しています。
