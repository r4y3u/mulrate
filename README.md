# MulRate v2.0.0 alpha.1

乗法に特化したレート制トレーニングアプリです。v1.0.0 beta.19を基礎に、ランキング・対戦へ進むための第1段階を実装しています。

## alpha.1の主な更新

- ローカルプロフィールを追加。
  - 12文字以内のニックネーム。
  - 端末内プレイヤーIDの自動生成。
  - 将来のランキング公開に対する同意設定。
  - 学習データとは分けてプロフィールだけ初期化可能。
- 「記録・ランキング」画面を追加。
  - 現在レート、最高レート、学習済み数、完了セット数。
  - 最近8セットの記録。
  - サーバー未接続時は端末内記録のみを表示。
- ランキング送信用データ構造を追加。
  - セッションID、開始・終了日時、所要時間。
  - 10問分の問題・回答・時間・正誤。
  - 問題列の簡易ダイジェスト。
- `online-adapter.js` を追加。
  - 現在は通信しない安全な未設定アダプター。
  - バックエンド決定後、このインターフェースを差し替え可能。
- v1のlocalStorageデータをv2へ自動移行。

## 現在のオンライン動作

この版は外部サーバーへ一切送信しません。プロフィール画面で公開同意を設定しても、バックエンドが未設定の間は端末内保存だけです。

## 実行方法

`index.html` をブラウザで開きます。外部ライブラリは不要です。

GitHub Pagesへ配置する場合は、次のファイルを同じ階層へ置いてください。

- `index.html`
- `style.css`
- `app.js`
- `online-adapter.js`

## データ移行

v1が使用していた `mulrate_v1_0_0_beta18_state` を検出すると、`mulrate_v2_0_0_alpha1_state` へ自動移行します。レート、学習済み、設定、履歴を維持します。

## 開発資料

- `V2_BACKEND_CONTRACT.md`: ランキング用バックエンドの接続契約。
- `TEST_REPORT.md`: alpha.1で実施した検査。
- `CURRICULUM.md`: 100段階カリキュラム。
- `DESIGN.md`: UI設計方針。
- `BALANCE.md`: レート設計方針。
