# MulRate v1.0.0 beta.5

かけ算の計算力を鍛える、レート制のブラウザアプリです。GitHub Pages でそのまま公開できます。

## 実行方法

`index.html` をブラウザで開きます。外部ライブラリ、サーバー、ビルド処理は不要です。

## 今回の主な更新

- 画面を「ナビウィンドウ」「テンキーウィンドウ」の固定2分割へ変更
- タッチ連続入力時の拡大・選択を抑制
- 1セット内の重複問題を抑制
  - `3×4` と `4×3` は同一扱い
- 解答欄を問題中・答え直し中だけ表示
- リザルトのレート表示を `before / delta / after` で固定管理
- 4桁区切りのレート表示
- キーボード入力時も画面テンキーが押し込まれる演出を追加
- 入力数値の軽い強調演出を追加
- 正解時の光スプラッシュ、誤答時の画面揺れを追加
- 回答後に回答速度と獲得・ロス目安を表示
- リザルトに「変動」「レート」「ロス」を表示
- 手書きパッドの折りたたみ・展開に対応
- 手書きパッドのレイアウト / オーバーレイ表示に対応
- 手書きパッドとテンキーの上下入れ替えに対応
- A〜F出題パターン遷移を強化
- 自動進行判定を追加
- 学習済み類型の反復練習を追加

## 保存データ

ブラウザの `localStorage` に保存します。個人情報、アカウント、外部通信は使いません。

保存キーは `mulrate_v1_0_0_beta2_state` です。

## 公開方法

GitHub リポジトリ直下にファイルを配置し、GitHub Pages を有効化してください。


## beta.3 更新

- 10問回答後、答え直し対象がない場合は自動で最終リザルトへ進むよう修正。
- テンキー名を「デフォルト」「疑似テンキー左」「疑似テンキー右」に整理。
- デフォルトの「C」を「←」表示に変更し、全テンキーで1文字削除に統一。
- 疑似テンキー左・右をキーボード配列に合わせて小型化・段差調整。
- Backspaceは1文字削除、Enter / Spaceは決定として常時利用可能。

## beta.4 更新

- 10問完了後、誤答がない場合は自動で最終リザルトへ進み、レートを即時確定するよう修正。
- 解答欄の表示切替関数を追加し、リザルト遷移中に表示が止まる問題を修正。
- 終了済みセットをホーム操作で破棄しないよう、未確定時は内部でレート確定する保護処理を追加。
- 疑似テンキー右の物理キー対応を整理。`;` / `Enter` / `Space` は決定、`Backspace` は1文字削除。

## beta.5 更新

- リザルト画面では内部進行処理を直接説明せず、「十分に定着」「次の確認へ進む」という表現に変更。
- 学習済み解放条件を補正。基本確認を十分な速度で終えた場合に加え、次類型へ進んだ既存データでも通過済み類型を復元します。
- 既存の localStorage データを読み込んだ際、通過済み類型が学習済みに反映されるよう移行処理を追加。
