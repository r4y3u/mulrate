# MulRate v2.0.0 alpha.1 テスト報告

## 実施済み

- `app.js` と `online-adapter.js` のJavaScript構文検査。
- HTMLのID重複検査。
- JavaScriptから参照する静的DOM IDの存在検査。
- DOMスタブ上での初期化検査。
- Chromium実行環境でのデスクトップ・スマホ画面検査。
- プロフィール画面への遷移、ニックネーム整形、公開同意保存。
- 記録・ランキング画面への遷移と数値表示。
- プロフィール・ランキング表示時に不要なテンキー領域が非表示になること。
- スタート操作から問題画面への遷移。
- 10問を全問正解したセットの完了処理。
- 履歴へのセッション検証データ保存。
- ランキング送信用データへの検証データ反映。
- v1保存キーからv2保存構造への移行ロジック確認。

## 確認結果

- 10問完了後、履歴1件と10問分の検証項目が生成された。
- プレイヤーID、セッションID、問題ダイジェストが生成された。
- バックエンド未設定時は `online.enabled` が有効にならず、外部送信しない。

## 実機で追加確認したい項目

- iPhone Safari / Braveでプロフィール入力欄が見切れないこと。
- スマホ縦画面でホームの5ボタンが押しやすいこと。
- v1を使用済みの実端末でレート・学習済み・設定が維持されること。
- 最近の記録が8件を超えた場合のスクロール。
